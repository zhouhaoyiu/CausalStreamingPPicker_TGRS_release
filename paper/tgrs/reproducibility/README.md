# CausalStreamingPPicker TGRS Reproducibility Notes

This folder records the reusable metadata for the TGRS supplement archive.

The supplement provides source code, model checkpoint metadata, sample replay data, evaluation scripts, figure sources, and precomputed evidence outputs. Raw third-party waveform records are not redistributed. Public waveforms should be obtained from their original providers, and the archive provides record identifiers, scripts, and manifests needed to reproduce the reported checks.

Use the top-level `README.md` inside `CausalStreamingPPicker_TGRS_supplement.zip` for the file manifest and SHA256 values.
